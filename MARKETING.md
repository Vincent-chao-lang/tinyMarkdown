## 🎯 为什么选择 tinyMarkdown？

### 📱 填补系统空白
iOS 系统默认不支持 Markdown 预览，tinyMarkdown 填补了这个空白，让你的手机能够：
- ✅ 在 Files App 中打开 .md 文件
- ✅ 看到格式完美的渲染内容
- ✅ 享受舒适的阅读体验

### 🎯 极致纯粹
- **只读设计**：专注于阅读，不编辑、不干扰
- **零网络依赖**：完全离线运行，无需联网
- **无隐私收集**：你的文档永远留在设备本地

### 🔒 隐私优先
```
❌ 不上传任何数据
❌ 不收集用户信息
❌ 不分析使用习惯
✅ 100% 本地处理
```

### 📄 完美支持
- 支持 `.md` 和 `.markdown` 文件
- 渲染标准 Markdown 语法：标题、列表、粗体、斜体、代码块、链接、图片
- 文件大小限制 1MB，保持应用轻量快速

### 🎨 简洁优雅
- 干净的界面设计
- 可调节字体大小（小/中/大/超大）
- 一键复制全部内容
- 长按选择部分文本复制

---

## 🚀 使用超简单

### 方式一：从 Files App 打开
1. 在 Files App 中找到你的 `.md` 文件
2. 长按文件，选择「打开方式」
3. 选择 tinyMarkdown
4. 享受完美渲染的 Markdown 内容！

### 方式二：应用内打开
1. 打开 tinyMarkdown
2. 点击「📂 打开文件」
3. 从文档选择器中选择文件
4. 立即开始阅读
